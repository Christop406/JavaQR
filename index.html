Home

Java is known as an Object-Oriented programming Language. As such, to program in Java you must be familiar with many aspects of Object-Oriented design, including but not limited to variables, methods, classes, and inheritance. This website offers an introduction to these concepts and links to find other sources of information and more advanced topics.

Getting Started

If you're a beginner, it can be hard to choose a good IDE to get started with your adventures in Java.

Which IDE is right for me?

When choosing an IDE, it is important that you know your own skills so that you can pick the one that would work best for you. An IDE with more features would be better suited for a person with an advanced knowledge of Java, while one with less and/or simple features might be more suited for a beginner of the language.

Most Common IDEs

BlueJ

BlueJ is a great IDE for beginners. It offers an easy way to write and test code before implementing it, including a code pad that lets you test variable assignations and many other things. No main classes or methods are needed to run programs. It does, however, lack any serious development features, such as Android publishing or very good project management. BlueJ is best for someone who would like to jump into Java quickly, without being roadblocked by the IDE's learning curve.

Eclipse

Eclipse is often the IDE of choice for experienced programmers. With a focus on the developer, it offers many of the tools you need to begin coding and many more. Eclipse can, at times, be unstable and is one of the harder IDEs to set up. Using Eclipse should be the advanced programmers or those with experience in Java programming. Due to its somewhat complicated setup process (if you don't already have Java) it can be a bit cumbersome to get started working in Eclipse.

IntelliJ Idea!

IntelliJ Idea! can best be described as a more-polished version of Eclipse. This means that it has a lot of the same features as Eclipse (as most Java IDEs do) but that it looks a bit cleaner and crashes a bit less. Google has chosen this IDE to be the development environment of choice for Android programming, which gives it a lot more importance in the industry. Like eclipse, this IDE is more for advanced users, but is a bit easier to pick up. It has great project management and is one of the best.
