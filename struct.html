Structure

Java is a Strict Type Language. This means every variable, method and class has to be explicitly defined a certain type.

There are some different Primitve Types within Java. A primitive type is a type of variable that does not conform to any interface or class and therefore cannot make use of any methods.

Structure of a Java Program

Classes

Classes, otherwise known as objects, are containers for methods and variables that have a close relationship to one another in a way that makes sense to the program and other programmers.
Every method and variable in Java must be encapsuled by a class or interface.

Methods

Methods allow a programmer to embed repeated or lengthy amounts of code inside a function that exectues when it is called. Methods can do everything from a simple math calculation to large algorithms to displaying colors on a computer screen. The only limit to a method's actions is the programmer's imagination.

Variables

Variables store data in the program. When a variable is declared, it gets a little spot in the computer's RAM and keeps that variable until it is no longer needed or reset.

Basic Structure of a Java Class

A Basic Java class consists of three parts:

  • The Accessibility Identifier (public or private)
  • The class declaration.
  • The name of the class <name>

    Structure of Java Methods

    Java's methods all share a similar design pattern to one another. To declare a method, one must include:

  • An Accessibility Identifier
  • A return type. A return type can be any object or primitive type.
  • A method name: exampleMethod in the example below.
  • Input parameters surrounded by parentheses. (int input)
    • public int exampleMethod(int input){
         return input;
    }

    The above code takes in an int (named input) as an input parameter. It returns the number that was passed in by the method call.

    Comments

    Comments help the developer keep track of what he/she is doing when coding. They can say anything in a program and are completely skipped by Java's compiler. To denote a comment in your code, simply add the characters // before you type, or for a multiline comment, use /*to start it and */ to end it, respectively.
